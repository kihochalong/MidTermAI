# MidTermAI

A Qt-based application with an optimized user interface.

## Features
- Clean, single-window interface
- Large text editing area (860x660 pixels)
- Responsive design with minimum size constraints
- Efficient layout with proper margins

## Building
This is a Qt application. To build:
```bash
qmake MidTermAI.pro
make
```

## Window Specifications
- Default size: 900x700 pixels
- Minimum size: 600x400 pixels
- Text edit area with 20-pixel margins
